# Netflix Global Subscription Pricing & Revenue Analysis

#test

## 📋 Project Overview
Analysis of Netflix subscription pricing 
across 65 countries and user behavior 
patterns across 10,000 users.

## 🎯 Research Questions
1. Which countries pay most and least for Netflix?
2. Which subscription plan generates most revenue?
3. Does higher price mean bigger content library?
4. Which global region has highest revenue potential?

## 🔑 Key Findings
- Turkey pays $3.00 vs Liechtenstein $20.46 
  — 7x price difference for same service!
- Correlation between price and library 
  size = -0.06 — almost no relationship!
- Standard & Premium plans = 70% of all users
- Europe has highest revenue potential globally
